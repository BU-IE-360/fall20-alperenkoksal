Forecasting the Total Number of Newly Established Firms¶

                                                                                                                                                                           Alperen Köksal

Introduction¶

The aim of this work is the forecasting the total number of newly established firms in Turkey. To do this, time series of total number of newly established firms from 2013 to 2020 is selected and analyzed. A model is created by using some independent variables which affect the behavior of the time series data, some seasonality factors which expalins the seasonal behavior of time series data and some special months such as: coup attempt and corona. With this model missing value of year 2020 which corresponds to twelfth month of year 2020 is predicted. This prediction will be checked when new data is added by Electronic Data Delivery System.

Conclusion¶

In order to have a good model, autocorrelation function and plots of time series data is analyzed in order to observe the seasonality factors and these seasonality factors, independent variables which is known as the affecting factor from knowledge or is observed as the affecting factor from plots and trend component which is also observed from plots are used in model.

There are still some residuals which is very high or very low in value. These points should be analyzed further and added to model in order to get a better model than the current model.

Prediction of twelfth month of 2020 is 9638.096. It will be checked when electronic data delivery system adds the next month's data.
